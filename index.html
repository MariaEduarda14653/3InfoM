Versionamento

O GIT É UM SOFTAWARE QUE PERMITE REALIZAR VERSÕES DE UM PROJETO. O GITHUB DESKTOP É UM SOFTWARE QUE FAZ A LIGAÇÃO ENTRE VSCODE, GIT E GITHUB.

O GitHub desktop permite fazer a sincronia entre o repositório local e o repositório remoto

Commit

É o processo de confirmação de uma versão,ou seja, a cada operação é possível confirmar a alteração

Push

Esse comando envia (sincroniza as alterações realizadas no repositório local com o repositório remoto no github)
